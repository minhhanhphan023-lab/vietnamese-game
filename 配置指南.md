# Google Sheets 配置 - 简化步骤

## 第一步：创建电子表格（2分钟）

1. 打开浏览器访问：https://sheets.google.com
2. 点击左上角的 "+" 创建新表格
3. 表格名称改为：越南语游戏成绩

## 第二步：设置工作表

1. 底部默认有"工作表1"，双击重命名为：关卡一
2. 点击底部的 "+" 添加新工作表，命名为：关卡二
3. 在"关卡一"工作表中：
   - A1 输入：姓名
   - B1 输入：得分
   - C1 输入：完成时间
4. 在"关卡二"工作表中重复第3步

## 第三步：创建脚本（3分钟）

1. 点击顶部菜单：扩展程序 → Apps Script
2. 会打开新页面，删除所有默认代码
3. 复制粘贴下面的代码（见 script.js 文件）
4. 点击顶部的保存图标（💾）
5. 点击"部署" → "新建部署"
6. 点击"选择类型"旁边的齿轮图标 ⚙️
7. 选择"网络应用"
8. 设置：
   - 说明：越南语游戏API
   - 执行身份：我
   - 访问权限：任何人
9. 点击"部署"
10. 点击"授权访问"
11. 选择你的Google账号
12. 点击"高级" → "转至xxx（不安全）"
13. 点击"允许"
14. 复制生成的网址（重要！）

网址格式类似：
https://script.google.com/macros/s/AKfycbxxx.../exec
